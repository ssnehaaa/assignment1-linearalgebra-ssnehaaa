# Linear Algebra Assignment

This assignment covers matrix transformations and vector similarity using NumPy.

---

## How to Submit

### Step 1 — Open the Notebook in Google Colab
1. Go to [colab.research.google.com](https://colab.research.google.com)
2. Click **File → Open Notebook → GitHub tab**
3. Paste your repository URL and select `linear_algebra_assignment.ipynb`

### Step 2 — Complete the Assignment
- For **written questions**: double-click the answer cell and type your response
- For **coding questions**: write your solution in the code cell provided
- Run all code cells before submitting (`Shift + Enter` to run a cell)

### Step 3 — Submit
Click the **"Save in GitHub to keep changes"** button at the top of Colab.

---

## Important Notes
- Do **not** modify the `# Input — do not modify` cells
- Make sure all code cells run without errors before submitting
- You can save your progress anytime using the button at the top — don't wait until the end.
